ğŸ”¹ Basic Page Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    Content here
</body>
</html>

What it means:

  • <!DOCTYPE html> tells the browser this is an HTML5 document.
  • <html> the whole webpage.
  • <head> invisible stuff: title, CSS, meta.
  • <body> visible content.

You should: Always start every HTML file with this structure.

ğŸ”¹ Text Elements

Headings

<h2>Section Title</h2>

Meaning: h1 is the most important title, h6 is the least.

You should: Have only ONE <h1> per page.

Paragraphs

<p>This is a paragraph.</p>

You should: Put normal text inside <p> tags.

Text Formatting

<strong>bold text</strong>
<em>italic text</em>
<br> line break

You should: Use these to highlight important information.
