HTML Syntax & Possibilities

A beginner-friendly explanation of how HTML works.

What is HTML?

HTML (HyperText Markup Language) is the language used to build the structure of web pages. It consists of elements called tags.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    Page content goes here.
</body>
</html>

Common HTML Tags

1. Headings

<h1>Main Title</h1>
<h2>Subtitle</h2>
<h3>Section Heading</h3>

2. Paragraphs

<p>This is a paragraph.</p>

3. Links

<a href="page.html">Click Me</a>

4. Images

<img src="image.jpg" alt="description">

5. Lists

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

6. Div & Span (containers)

<div>Block container</div>
<span>Inline container</span>

7. Inputs & Forms

<form>
    <input type="text" placeholder="Your name">
    <button>Submit</button>
</form>
